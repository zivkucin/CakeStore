# Cake Shop Desktop Application

##  Overview  
Cake Shop is a simple desktop application developed using **C# and WPF**.  
The application allows basic management of products (cakes) through a user-friendly interface and database integration.

---

## Technologies  
- C#  
- .NET (WPF)   
- SQL Database  

---

## Features  
- Add, edit, and delete products  
- Display product list  
- Basic data validation  
- Database integration  

---

## Implementation  
- Implemented basic CRUD operations  
- Used data binding for dynamic UI updates  
- Organized application using standard WPF structure
